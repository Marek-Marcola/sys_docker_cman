docker cman
===========

Docker container management tools.

Deployments models: docker, podman, systemd, pacemaker

Install
-------
Install:

    ./cman.sh --install

    cp -fv cman.sh /usr/local/bin/cman.sh
    cp -fv cman.sh /usr/local/bin/cman-exec.sh

Verify:

    cman.sh --version

Help:

    cman.sh --help
